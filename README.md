## How To Use Expense Tracker CLI

### Adding an Expense
```bash
expense-cli add --description {expense} --amount {100}
```

### List Expenses
```bash
expense-cli list
```

### Expense Summary
```bash
expense-cli summary
```

#### Expense Summary By Month
```bash
expense-cli summary --month 8 
```

### Delete Expense
```bash
expense-cli delete --id 2
```
